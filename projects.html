Imitation Learning for Baxter using Vicon

GitHub Repository Teleoperation Interface
GitHub Repository HMM Training
Vicon Motion Capture system is used for teleoperating the Baxter Robot and VR Headset is used to remove visual correspondance issue. Hidden Markov Model (HMM) is used to learn different Trajectories for different actions
Contribution: Created an interface to remotely teleoperate baxter robot. Collected trajectories using the interface as kinesthetic teaching is hard to move in precise motions. Baxter SDK (by rethink robotics) is used for Torque control. High Level position control is implemented to track the trajectory (runs at 200 Hz). HMM is trained in Matlab using Viterbi Algorithm. Bi-RRT* is used for dynamic replanning.

Autonomous Infrastructure Inspection


Used DJI Matrice 100 along with Hukoyu UTM 30 LX Lidar with onboard computer Odroid U3 (1.7Ghz Quad core, 2GB RAM, 64GB EMMC Storage), for the purpose of Infrasture Inspection using UAV.
Contribution: Coverage Planner Development and Implementation. Given a 3D point cloud map, a 3D plan automatically generated to inspect full map, considering the parameters like position uncertainty (because of not properly tuned controller, also turbulence and wind effect) and camera instrinsics. This uncertainty is emperically modelled and used to generate the path plan (which includes velocity of the robot, distance from the wall, and 3D Path) to inspect in less flight time with more coverage output.
Real-time monitoring of coverage and replan is generated using RRT (local plan) and Travelling Salesman Problem (global plan with waypoints)
